# 창대교회 웹사이트

PHP와 Tailwind CSS를 기반으로 제작된 창대교회 공식 웹사이트입니다. 반응형 디자인과 다크 모드를 완벽하게 지원하며, 통합된 관리자 대시보드를 통해 교회 콘텐츠를 효율적으로 관리할 수 있습니다.

## ✨ 주요 특징 (Features)

- **반응형 UI/UX**: Tailwind CSS를 활용하여 모바일, 태블릿, 데스크탑 환경에 최적화된 인터페이스를 제공합니다.
- **다크 모드 지원**: 시스템 설정 감지 및 사용자 수동 토글(헤더 버튼)을 통해 라이트/다크 모드를 전환합니다 (`localStorage` 저장).
- **콘텐츠 관리 시스템 (CMS)**:
  - **통합 대시보드**: 게시물 목록 조회와 등록/수정 작업을 한 페이지에서 처리하여 작업 효율을 높였습니다.
  - **게시판 관리**: 소식, 예배(설교), 주보, 칼럼, 특별영상 등 5개 섹션 관리.
  - **강력한 에디터**: Quill.js를 연동하여 이미지, 비디오, 텍스트 서식을 손쉽게 작성할 수 있습니다.
  - **주보 관리 특화**: 다중 이미지 업로드 지원 및 `Sortable.js`를 이용한 드래그 앤 드롭 이미지 순서 변경 기능을 제공합니다.
  - **보안**: 관리자 로그인 세션 처리 및 비밀번호 변경 기능 제공 (`admin_config` 테이블 사용).
- **동적 페이지 라우팅**: `index.php` 파일이 URL 파라미터(`?page=...`)를 분석하여 적절한 페이지 콘텐츠를 동적으로 로드합니다.
- **미디어 최적화**:
  - YouTube 링크 자동 임베드 변환.
  - 주보 이미지 클릭 시 확대 모달(Lightbox) 기능 (`assets/js/main.js`).
  - AOS (Animate On Scroll) 라이브러리를 이용한 스크롤 애니메이션.

## 🛠️ 기술 스택 (Tech Stack)

- **Backend**: PHP
- **Database**: MySQL (PDO)
- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Libraries**:
  - Quill.js: 관리자 페이지 텍스트 에디터
  - Sortable.js: 관리자 페이지 이미지 순서 정렬
  - AOS: 스크롤 애니메이션
  - Kakao Map SDK: 오시는 길 지도 표시

## 📄 파일별 주요 역할

### `html/index.php`

- 모든 사용자 요청의 진입점입니다.
- `?page=` 쿼리 스트링을 받아 `pages/` 폴더에서 해당 파일을 불러옵니다.
- `header.php`와 `footer.php`를 포함하여 전체 페이지의 레이아웃을 구성합니다.

### `html/admin/`

- **`index.php`**: 관리자가 로그인 후 보게 되는 메인 화면입니다. 좌측에는 글쓰기/수정 에디터가, 우측에는 기존 글 목록이 표시됩니다.
- **`process.php`**: 모든 백엔드 데이터 처리를 담당하는 핵심 파일입니다. 폼에서 전송된 데이터를 받아 DB에 저장, 수정, 삭제하는 로직을 수행합니다.

### `html/includes/`

- **`db.php`**: 데이터베이스 연결 정보를 담고 있으며, PDO 객체를 생성하여 다른 파일에서 사용할 수 있게 합니다.
- **`functions.php`**: 여러 페이지에서 공통으로 사용되는 함수들을 정의합니다.
  - `getBoardPost()`: 특정 게시물의 현재/이전/다음 글 정보를 한 번에 가져오는 중요한 함수입니다.
- **`header.php`**: 모든 페이지 상단에 표시되는 로고, 네비게이션 메뉴, 다크 모드 토글 버튼 등을 포함합니다.

### `html/pages/`

- 각 페이지의 실제 콘텐츠를 담고 있는 파일들입니다.
- `index.php`에 의해 동적으로 로드되며, `?sub=` 파라미터에 따라 한 파일 내에서 여러 하위 메뉴의 내용을 분기하여 처리하기도 합니다.

## 🚀 시작하기 (Getting Started)

1.  **데이터베이스 설정**:
    - `includes/db.php` 파일에 자신의 데이터베이스 연결 정보를 입력합니다.
    - 아래 테이블들을 데이터베이스에 생성합니다.
      - `users`: 관리자 계정 정보
      - `news`: 교회소식
      - `sermon`: 주일예배
      - `bulletin`: 주보
      - `column`: 목회칼럼
      - `videos`: 특별영상

2.  **서버 설정**:
    - 웹 서버의 Document Root를 `html/` 디렉토리로 지정합니다.
    - `html/uploads/` 디렉토리의 파일 쓰기 권한을 허용합니다 (예: `755` 또는 `777`).

3.  **관리자 페이지 접속**:
    - `http://[your-domain]/admin/login.php` 로 접속하여 로그인합니다.
    - 초기 관리자 계정은 `users` 테이블에 직접 추가해야 합니다. (비밀번호는 `password_hash()` 함수로 해시하여 저장)
